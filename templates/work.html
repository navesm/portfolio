001 : Facial Recognition App

This app was built using React for the front end, a Node.js server, and a PostgreSQL database. Once a user registers with an email and password, you can enter any image URL address and it will display the image with any people's faces being detected!

Make sure you enter a URL and not a base64 or else it won't be able to detect the faces!

Also, the server and database for the app is hosted on Heroku, so there will be a slight delay as the server wakes up on first use. It can take a little while to spin up, so please bear with it! After that initial start-up and sign-in, it is plenty quick and responsive!
