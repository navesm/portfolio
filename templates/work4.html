004 : Heart Disease Classification Model

This binary classification model was made using a dataset from Kaggle of over a thousand patients medical records and a determination of whether they had heart disease or not.

The model was trained using the Scikit-Learn library and the Random Forest Classification algorithm. The Jupyter Notebook can be found at the github repository below.
